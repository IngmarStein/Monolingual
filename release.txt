How to create a Monolingual release

1) Bump version number in
   - Info.plist
   - MonolingualHelper-Info.plist
   - InfoPlist.strings
   - Makefile
   - README.markdown
2) Add changelog to readmes
3) make release
4) Copy release/*.dmg to htdocs/files
5) Tag release
6) Update index.html
7) Update changelog.html
8) Update appcast.xml
9) Upload website (upload.sh)
10) Create release on GitHub
11) Announce release on http://www.macupdate.com
