  • Now requires OS X 10.7 and a 64-bit capable Mac (to leverage new technologies such as XPC and ARC).

Version 1.2.0

Almost completely rewritten and features a completely new user interface. Monolingual no longer uses a custom password panel but Mac OS X's Security Framework. The perl script is no longer needed by the application, but is still included in the Monolingual distribution. Many languages have been added. Requires OS X 10.2 or later.

Version 1.1.2

Adds German localization and documentation by Alex Thurley, and fixes a GUI bug that occurs when all languages except one are selected.

Version 1.1.1

Adds Swedish localization by Joel Arvidsson and Polish localization by Mariusz Ostrowski, as well as auto-detecting the current language and some minor GUI fixes.

Version 1.1

Adds support for removing all of the new languages introduced by Apple. Many people have inadvertantly added these to their system using Software Update. It also removes the of "two-letter country" resources to reclaim additional space.

Version 1.0.1

Adds Japanese localization by Takehiko Hatatani and an icon by Adam Betts, as well as fixing a problem in OS X 10.1.2 where an incorrect password causes the program to freeze.
